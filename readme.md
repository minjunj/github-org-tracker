### Toy Project

#### 조직에 누가 얼마나 기여했을까?

Github API를 활용하여 깃허브 조직의 커밋 횟수를 크롤링하여 시각화.

Github API에서는 Public repository만 수집할 수 있기에 private repository는 조회되지 않습니다.

실행 방법

1. .env 에서 변수를 세팅

2. streamlit run st.py 으로 실행
